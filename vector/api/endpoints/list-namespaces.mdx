---
title: "List Namespaces"
description: "This endpoint lists the namespaces of an index."
api: "GET https://{endpoint}/list-namespaces"
authMethod: "bearer"
---

## Request

This endpoint doesn't require any additional data.

## Response

<ResponseField name="namespaces" type="string[]">
  The list of namespace names.
</ResponseField>

<RequestExample>

```sh curl
curl https://better-dodo-20522-us1-vector.upstash.io/list-namespaces \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```js Node
const url = "https://better-dodo-20522-us1-vector.upstash.io/list-namespaces"; // Replace with your index endpoint.
const token = "YOUR_TOKEN"; // Replace with your actual token

fetch(url, {
  headers: {
    Authorization: `Bearer ${token}`,
    "Content-Type": "application/json",
  },
})
  .then((response) => response.json())
  .then((data) => console.log(data))
  .catch((error) => console.error("Error:", error));
```

</RequestExample>

```json Response
{
  "result" : ["", "ns0", "ns1"]
}
```

</ResponseExample>
