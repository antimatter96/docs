---
title: "Delete Namespace"
description: "This endpoint deletes a namespace of an index."
api: "DELETE https://{endpoint}/delete-namespace/{namespace}"
authMethod: "bearer"
---

## Request

This endpoint doesn't require any additional data.

## Path

<ParamField path="namespace" type="string" required>
  The namespace to delete.
</ParamField>

## Response

<ResponseField name="Response" type="Object">
  Returns `"Success"` on successful upsert operation.
</ResponseField>

<RequestExample>

```sh curl
curl https://better-dodo-20522-us1-vector.upstash.io/delete-namespace/ns \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -X DELETE
```

```js Node
const url = "https://better-dodo-20522-us1-vector.upstash.io/delete-namespace/ns"; // Replace with your index endpoint.
const token = "YOUR_TOKEN"; // Replace with your actual token

fetch(url, {
  headers: {
    Authorization: `Bearer ${token}`,
    "Content-Type": "application/json",
  },
  method: "DELETE",
})
  .then((response) => response.json())
  .then((data) => console.log(data))
  .catch((error) => console.error("Error:", error));
```

</RequestExample>

<ResponseExample>
```json Response
{
  "result" : "Success"
}
```

</ResponseExample>
